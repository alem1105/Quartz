Java è un linguaggio compilato, infatti noi scriviamo un **source code** ad alto livello e non linguaggio macchina che verrà poi compilato e tradotto.
Quello che accade di solito però con i linguaggi compilati è che la compilazione è specifica per il sistema operativo, con Java invece abbiamo uno **step intermedio**:
**Source Code** -> **Compiler** -> **Byte Code** -> **JVM** -> **Machine Code**
Il Byte Code è un linguaggio letto da tutti i sistemi operativi che viene letto non direttamente dal sistema ma dalla **JVM (Java Virtual Machine)** che lo tradurrà per il sistema operativo.

**Note:**
- **JVM:** Java Virtual Machine, contenuta nel **JDK**
- **JDK:** Java Development kit che contiene **JRE**
- **JRE:** Java Runtime Environment che contiene librerie e toolkits

Il **JDK** è il kit quindi che contiene tutto il necessario per compilare ed eseguire codice Java.
